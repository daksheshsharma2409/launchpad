# Launchpad 🚀

> **Every Opportunity. One Feed. Built for You.**

Launchpad is a smart opportunity aggregator that brings hackathons, workshops, cultural fests, internships, and jobs into a single personalized platform — so students never miss something they actually care about.

---

## 🎯 The Problem

Opportunities are scattered everywhere — Devpost, Unstop, Internshala, LinkedIn, Devfolio, Eventbrite. Students either:

- Miss deadlines because they didn't know about the opportunity
- Spend hours hopping between platforms every day
- Get overwhelmed by irrelevant listings

**Launchpad fixes this.** One feed, personalized to your skills, branch, and interests.

---

## ✨ Features

### 🏠 Personalized Feed

- "For You" feed tailored to selected interests
- Trending and Newly Added sections
- Deadline countdown badges on every card
- "Closing Soon" and "New" labels at a glance

### 📂 Opportunity Categories

- 🏆 **Hackathons** — competitive coding and innovation challenges
- 🛠️ **Workshops** — skill-building sessions and bootcamps
- 🎭 **Cultural Fests** — college fests and creative competitions
- 💼 **Internships** — short-term industry experience roles
- 👔 **Jobs** — entry-level full-time opportunities

### 📌 Opportunity Cards

- Organization logo, title, tags, deadline, stipend/prize, and location
- Quick Save button for bookmarking
- Direct "Apply" button linking to the original source
- Share opportunity link
- Visual badges — Closing Soon / New / Featured

### 🔍 Search & Filter

- Search opportunities by keyword
- Filter by category, location, and remote/in-person

### 🎯 Onboarding

- Select your branch, year of study, skills, and interests
- Feed adjusts to show relevant opportunities first

### 💾 Saved Opportunities

- Bookmark opportunities to revisit later
- Dedicated saved page to manage your list

### Installation

```bash
git clone https://github.com/your-username/launchpad.git
cd launchpad
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the app.
